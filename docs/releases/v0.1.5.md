# AudioMate v0.1.5 Release Notes

This patch adds excluded trigger apps.

## Added

- New Excluded Apps settings page.
- Scan flow can add unknown audible apps either as music/BGM targets or as excluded trigger apps.
- Excluded trigger apps do not cause music/BGM ducking when they are audible.
- Custom excluded trigger apps can be removed from settings.
