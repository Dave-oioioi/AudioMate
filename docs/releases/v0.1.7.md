# AudioMate v0.1.7 Release Notes

This patch adds branding polish and safer overwrite installs.

## Added

- New simple iOS-style AudioMate tray and application icon.
- Generated multi-size `.ico` plus source PNG assets.
- Installer icon now uses the AudioMate icon.
- Installer detects existing installs through the stable AppId.
- Installer terminates a running `AudioMate.App.exe` before overwrite installation.
- Installer is configured for Inno Setup application closing support.

## Verification

```powershell
dotnet test AudioMate.sln --configuration Release
powershell -ExecutionPolicy Bypass -File scripts\publish-release.ps1 -Version 0.1.7
```
