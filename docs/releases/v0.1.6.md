# AudioMate v0.1.6 Release Notes

This patch makes app selection interactive inside Settings.

## Added

- Music/BGM Apps page can scan currently audible apps and add selected items.
- Music/BGM Apps page can manually add a process name.
- Excluded Apps page can scan currently audible apps and add selected items.
- Excluded Apps page can manually add a process name.
- Shared process selection dialog with multi-select checkboxes.

## Verification

```powershell
dotnet test AudioMate.sln --configuration Release
powershell -ExecutionPolicy Bypass -File scripts\publish-release.ps1 -Version 0.1.6
```
