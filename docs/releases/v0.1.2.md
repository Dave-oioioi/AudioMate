# AudioMate v0.1.2 Release Notes

This patch smooths volume changes so ducking no longer feels abrupt.

## Added

- Fade-out when music/BGM is lowered.
- Fade-in when music/BGM is restored.
- Settings for fade-out and fade-in duration in the mini settings window.

## Defaults

- Fade-out: 180 ms.
- Fade-in: 260 ms.
- Both durations can be adjusted from 0 to 3000 ms.
