# AudioMate v0.1.9

## Changes

- Changed app selection from audible-session-only scanning to PC application scanning.
- Scans running processes, Windows App Paths, and common install folders for executable process names.
- Added search to the application selection dialog so manual typing works as an auxiliary search flow.
- Updated the tray action and settings buttons to use “扫描PC应用”.

## Build

powershell -ExecutionPolicy Bypass -File scripts\publish-release.ps1 -Version 0.1.9
