# AudioMate v0.1.4 Release Notes

This patch adds microphone input ducking.

## Added

- Default microphone input level polling.
- Music/BGM ducking while microphone input is above the configured threshold.
- Microphone input threshold setting in the Audio Behavior page.
- Background recovery when microphone scanning fails because no capture device is available or the device is switching.
