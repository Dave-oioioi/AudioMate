# AudioMate v0.1.0 Release Notes

AudioMate v0.1.0 is the first MVP release candidate for the Windows tray-first audio assistant.

## Included

- Tray app with enable/pause, mode switching, quick duck volume controls, Codex narration toggle, startup toggle, audible app scanning, and mini settings.
- JSON configuration stored under `%APPDATA%\AudioMate\config.json`.
- File narration queue under `%LOCALAPPDATA%\AudioMate\NarrationQueue`.
- Codex/Aural bridge script that queues requests when AudioMate is running and falls back to standalone Aural speech when it is not.
- NAudio-backed Windows render session scanning and music app duck/restore.
- Background narration worker that pre-ducks music while speaking and restores after the configured delay.
- Custom music app confirmation from audible app scan results.
- HKCU startup registration from the tray setting.
- Automatic ordinary-audio ducking: non-music render sessions above the trigger threshold now lower configured music apps.
- Chinese tray menu, settings window, prompts, and notifications.

## Known Gaps

- Microphone trigger detection is not yet implemented.
- Fade in/out settings are persisted but volume changes are currently immediate.
- Installer-driven Codex Aural Skill installation is still a skeleton.
- No signed installer yet.
