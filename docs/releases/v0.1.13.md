# AudioMate v0.1.13

## Changes

- Added single-instance startup protection so only one AudioMate tray process can run at a time.
- A second launch exits immediately instead of creating another tray icon/background worker.
- Updated installer/release defaults to build `AudioMate-Setup-v0.1.13.exe`.

## Build

```powershell
powershell -ExecutionPolicy Bypass -File scripts\publish-release.ps1 -Version 0.1.13
```
