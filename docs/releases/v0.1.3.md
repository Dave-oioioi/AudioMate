# AudioMate v0.1.3 Release Notes

This patch improves the mini settings window layout and usability.

## Changed

- Replaced the cramped single-panel settings UI with tabbed pages.
- Split settings into Music/BGM Apps, Audio Behavior, and Codex Narration.
- Added a visible custom music/BGM app list for scan-added targets.
- Added removal support for custom music/BGM targets.
- Replaced the duck volume slider with clearer numeric controls.
- Added units for percent and millisecond values.
- Made the settings window resizable with a larger default size.
