# AudioMate v0.1.11

## Changes

- Promoted Windows volume-mixer/audio-session applications as the first scan source for app recognition.
- Added source labels in the scan picker: `[音量合成器]` for mixer apps and `[PC应用]` for broader PC discovery.
- Kept saved settings clean by storing only normalized process names, not UI labels.
